# Campus Lost and Found Portal

A centralized web application for managing lost and found items within a college campus.  
Students and staff can report lost items, upload found items, search listings, and reclaim belongings efficiently.

---

## 🚀 Features

- User Authentication (Login/Register)
- Report Lost Items
- Report Found Items
- Upload Item Images
- Search & Filter Items
- Item Status Tracking
- Responsive UI
- Admin Management System
- Secure Backend APIs
---
## ⭐ Conclusion

The Campus Lost and Found Portal helps students recover lost belongings quickly through a centralized digital platform, improving campus communication and efficiency.
